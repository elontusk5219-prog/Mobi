## 使用说明（一个一个对）

- **先对 0**：大脑 Agent 完成后，再正式进 1。
- **1 和 2 一起对**：先对「画像 ↔ 进化/人格槽」契约；你做进化时按契约接 Mock 或真实画像。
- **3 可与 2 并行**：你做进化客户端时，画像服务（或 spec）可同步推进，减少互相等。
- **4**：独立小项，随时可插空对。
- **5–6**：行为 Agent 做表现，当前助手做「行为→画像」通道与约定；对表时可将 5、6 当一组对。
- **7–11**：按表顺序逐项对即可。
