## 📁 项目结构

```
/Volumes/cursor/Mobi/
├── Mobi/                    # iOS Swift 代码
│   ├── Mobi.xcodeproj
│   ├── Core/               # 核心：MobiEngine, MobiBrain, VisualDNA
│   ├── Features/           # 功能模块
│   └── Services/           # 服务：音频、记忆、进化
├── backend/                 # 后端代码
├── docs/                   # 文档（见上表）
├── Config/                 # 配置文件
└── scripts/                # 脚本（add_mobi_placeholder 等；资产生成管线在独立项目）
```
